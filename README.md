# Not-Hesaplama

    Bu bir üniversite not hesaplama uygulamasıdır
    
    ![Screenshot](https://user-images.githubusercontent.com/104618142/209575817-42570f29-c518-4a90-bfa5-1cf8c16e04e0.png)

    ortalamanızı hesaplayacaktır.
